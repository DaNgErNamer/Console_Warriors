# Console Warriors
Пошаговая игра на Unity на основе консольной игры от DangerName. 
Стек разработки: **Unity**, **Blender**, **Github**.

# Features
- Целевые платформы: Android, iOS и абстрактный ПК.
- WebGL уже не поддерживается из-за использования HDRP.
